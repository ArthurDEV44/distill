# Intégration Windsurf

Ce guide vous accompagne dans la configuration de Distill avec l'IDE Windsurf.

## Prérequis

- Windsurf installé ([télécharger](https://windsurf.com))
- Node.js 20+ installé

## Configuration rapide (30 secondes)

```bash
npm install -g distill-mcp
distill-mcp setup --windsurf
```

Terminé ! Redémarrez Windsurf et vous êtes prêt.

## Configuration manuelle

Si la configuration automatique n'a pas fonctionné :

**Via l'interface :** Dans le chat Windsurf, cliquez sur l'icône **marteau (MCP servers)** → **Configure** → **View raw config**

**Via fichier :** Éditez `~/.codeium/windsurf/mcp_config.json` (macOS/Linux) ou `%USERPROFILE%\.codeium\windsurf\mcp_config.json` (Windows) :

```json
{
  "mcpServers": {
    "distill": {
      "command": "distill-mcp",
      "args": ["serve"]
    }
  }
}
```

Ou avec npx (sans installation globale) :

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp", "serve"]
    }
  }
}
```

## Vérifier l'installation

Après avoir modifié la configuration, cliquez sur le bouton **Refresh** dans le panneau MCP servers, puis demandez :

```
Liste les outils MCP disponibles
```

## Dépannage

### MCP ne charge pas

1. Vérifiez l'emplacement du fichier de config : `~/.codeium/windsurf/mcp_config.json`
2. Validez la syntaxe JSON
3. Cliquez sur le bouton **Refresh** après modification
4. Exécutez `distill-mcp doctor` pour vérifier l'installation

### Erreurs de permissions

Sur macOS, vous devrez peut-être autoriser le terminal dans Préférences Système > Sécurité et confidentialité.
