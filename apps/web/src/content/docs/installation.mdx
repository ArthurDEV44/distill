# Installation

Dﾃｩmarrez avec Distill en moins d'une minute.

## Dﾃｩmarrage rapide (Recommandﾃｩ)

```bash
# Installer globalement
npm install -g distill-mcp

# Auto-configurer votre IDE
distill-mcp setup
```

L'assistant de configuration interactif vous guidera dans la configuration de vos IDEs (Claude Code, Cursor, Windsurf, Antigravity).

## Prﾃｩrequis

- **Node.js 20+** requis
- Un IDE compatible MCP (Claude Code, Cursor, Windsurf ou Antigravity)

## Options d'installation

### Option 1 : Installation globale + Auto Setup

```bash
npm install -g distill-mcp
distill-mcp setup
```

### Option 2 : Configurer un IDE spﾃｩcifique

```bash
distill-mcp setup --claude       # Claude Code uniquement
distill-mcp setup --cursor       # Cursor uniquement
distill-mcp setup --windsurf     # Windsurf uniquement
distill-mcp setup --antigravity  # Antigravity uniquement
```

### Option 3 : Exﾃｩcuter avec npx (Sans installation)

```bash
npx distill-mcp serve
```

Puis ajoutez manuellement ﾃ la config de votre IDE (voir ci-dessous).

## Configuration manuelle de l'IDE

Si vous prﾃｩfﾃｩrez une configuration manuelle ou si `distill-mcp setup` n'a pas fonctionnﾃｩ :

### Claude Code

Via CLI : `claude mcp add distill -- npx distill-mcp serve`

Ou ﾃｩditez `~/.claude.json` :

```json
{
  "mcpServers": {
    "distill": {
      "command": "distill-mcp",
      "args": ["serve"]
    }
  }
}
```

Ou avec npx (sans installation globale) :

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp", "serve"]
    }
  }
}
```

### Cursor

ﾃ嬰itez `~/.cursor/mcp.json` (global) ou `.cursor/mcp.json` (par projet) :

```json

  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["-y", "distill-mcp", "serve"]
    }
  }
}
```

Le flag `-y` permet ﾃ npx d'installer automatiquement le package si nﾃｩcessaire.

### Windsurf

ﾃ嬰itez `~/.codeium/windsurf/mcp_config.json` :

      "command": "npx",

Le flag `-y` permet ﾃ npx d'installer automatiquement le package si nﾃｩcessaire.

Le flag `-y` permet ﾃ npx d'installer automatiquement le package si nﾃｩcessaire.

## Rﾃｩfﾃｩrence CLI

Binaire : `distill-mcp`. Quatre commandes principales.

```bash
distill-mcp serve      # Dﾃｩmarre le serveur MCP sur le transport stdio
distill-mcp setup      # Configure les IDE et installe hooks/agents
distill-mcp doctor     # Diagnostique l'installation
distill-mcp analyze    # Analyse l'utilisation des tokens dans un codebase
distill-mcp --help     # Affiche toutes les options
distill-mcp --version  # Affiche la version installﾃｩe
```

### `distill-mcp serve`

| Flag | Effet |
|------|-------|
| `--verbose` | Active les logs dﾃｩtaillﾃｩs sur stderr (stdout reste le canal MCP) |

### `distill-mcp setup`

Cibler un IDE prﾃｩcis :

| Flag | Effet |
|------|-------|
| `--claude` | Configure uniquement Claude Code |
| `--cursor` | Configure uniquement Cursor |
| `--windsurf` | Configure uniquement Windsurf |
| `--antigravity` | Configure uniquement Antigravity |

Installer les extensions opt-in (introduites en `v0.10.0`) :

| Flag | Effet |
|------|-------|
| `--hooks` | Installe le hook de prﾃｩ-commit dans le projet courant |
| `--install-precompact-hook` | Ajoute le hook PreCompact dans `~/.claude/settings.json` (idempotent, atomique via tempfile + rename) |
| `--uninstall-precompact-hook` | Retire proprement l'entrﾃｩe PreCompact |
| `--install-agent` | Copie le template `distill-compressor.md` dans `~/.claude/agents/` |
| `--uninstall-agent` | Supprime l'agent installﾃｩ |

Options transverses :

| Flag | Effet |
|------|-------|
| `--dry-run` | Affiche les changements sans modifier le filesystem |
| `--user-dir=<chemin>` | Surcharge `HOME` pour localiser `settings.json` (utile en tests) |
| `--force`, `-f` | ﾃ営rase une configuration existante mﾃｪme si elle diffﾃｨre du template |

Voir la [page du marqueur et du hook PreCompact](/fr/docs/compressed-marker) pour les dﾃｩtails d'intﾃｩgration.

### `distill-mcp doctor`

Aucun flag. Vﾃｩrifie la version Node, la prﾃｩsence de `distill-mcp`, la configuration des IDE dﾃｩtectﾃｩs et l'ﾃｩtat des hooks installﾃｩs.

### `distill-mcp analyze`

| Flag | Effet |
|------|-------|
| `--patterns`, `-p` | Patterns glob ﾃ scanner (dﾃｩfaut `**/*.{ts,tsx,js,jsx,py,go,rs}`) |
| `--threshold`, `-t` | Seuil d'alerte en tokens (dﾃｩfaut `2000`) |
| `--json`, `-j` | Sortie JSON au lieu du rapport texte |
| `--output`, `-o` | ﾃ営rit le rapport dans un fichier au lieu de stdout |
