# Documentation Distill

Distill est un serveur MCP open source qui vous aide à optimiser l'utilisation des tokens LLM. Il fournit une compression intelligente du contexte, une lecture de fichiers optimisée et des outils d'analyse pour Claude Code, Cursor et Windsurf.

**Pas d'authentification, pas de clés API, pas de services cloud** - installez et utilisez directement.

> **Qu'est-ce que MCP ?**
> MCP (Model Context Protocol) est un standard ouvert qui permet aux assistants IA (Claude Code, Antigravity, Cursor, Windsurf) de communiquer avec des serveurs externes. Distill est un serveur MCP qui aide votre IA à consommer moins de "tokens" (unités de texte facturées) en compressant intelligemment les informations qu'elle traite.

## Démarrage rapide

Commencez avec Distill en 3 étapes simples :

### 1. Installer le serveur MCP

```bash
# Avec npm
npm install -g distill-mcp

# Ou exécuter directement avec npx
npx distill-mcp
```

### 2. Configurer votre IDE

Ajoutez le serveur MCP à la configuration de votre IDE. Pour Claude Code, éditez `~/.claude.json` :

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp"]
    }
  }
}
```

### 3. Commencer à économiser des tokens

Le serveur MCP fournit 3 outils toujours actifs : `auto_optimize`, `smart_file_read` et `code_execute`.

## Fonctionnalités

- **3 outils de precision** - Compression auto, lecture AST et SDK TypeScript sandboxe
- **Parsing AST** - Extraction de fonctions, classes et types depuis le code (TypeScript, Python, Go, Rust, PHP, Swift)
- **Support universel des IDE** - Fonctionne avec Claude Code, Cursor, Windsurf et tout client compatible MCP
- **50-95% d'économie de tokens** - Selon le type de contenu
