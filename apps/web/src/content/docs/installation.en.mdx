# Installation

Get started with Distill in under a minute.

## Quick Start (Recommended)

```bash
# Install globally
npm install -g distill-mcp

# Auto-configure your IDE
distill-mcp setup
```

The interactive setup wizard will guide you through configuring your IDEs (Claude Code, Cursor, Windsurf, Antigravity).

## Prerequisites

- **Node.js 20+** required
- An MCP-compatible IDE (Claude Code, Cursor, Windsurf, or Antigravity)

## Installation Options

### Option 1: Global Install + Auto Setup

```bash
npm install -g distill-mcp
distill-mcp setup
```

### Option 2: Configure Specific IDE

```bash
distill-mcp setup --claude       # Claude Code only
distill-mcp setup --cursor       # Cursor only
distill-mcp setup --windsurf     # Windsurf only
distill-mcp setup --antigravity  # Antigravity only
```

### Option 3: Run with npx (No Install)

```bash
npx distill-mcp serve
```

Then manually add to your IDE config (see below).

## Manual IDE Configuration

If you prefer manual setup or `distill-mcp setup` didn't work:

### Claude Code

Via CLI: `claude mcp add distill -- npx distill-mcp serve`

Or edit `~/.claude.json`:

```json
{
  "mcpServers": {
    "distill": {
      "command": "distill-mcp",
      "args": ["serve"]
    }
  }
}
```

Or with npx (no global install):

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp", "serve"]
    }
  }
}
```

### Cursor

Edit `~/.cursor/mcp.json` (global) or `.cursor/mcp.json` (per-project):

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["-y", "distill-mcp", "serve"]
    }
  }
}
```

The `-y` flag allows npx to automatically install the package if needed.

### Windsurf

Edit `~/.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["-y", "distill-mcp", "serve"]
    }
  }
}
```

The `-y` flag allows npx to automatically install the package if needed.

### Antigravity

Open Antigravity, click **Agent session** > **"..."** menu > **MCP Servers** > **Manage MCP Servers** > **View raw config**, then edit `mcp_config.json`:

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["-y", "distill-mcp", "serve"]
    }
  }
}
```

The `-y` flag allows npx to automatically install the package if needed.

## Verify Installation

```bash
# Check everything is configured correctly
distill-mcp doctor
```

Then restart your IDE and ask: "Use auto_optimize to compress this build output"

## CLI Reference

Binary: `distill-mcp`. Four main commands.

```bash
distill-mcp serve      # Start the MCP server on stdio transport
distill-mcp setup      # Configure IDEs and install hooks/agents
distill-mcp doctor     # Diagnose installation
distill-mcp analyze    # Analyze token usage in a codebase
distill-mcp --help     # Show all options
distill-mcp --version  # Print installed version
```

### `distill-mcp serve`

| Flag | Effect |
|------|--------|
| `--verbose` | Enable verbose logging on stderr (stdout stays the MCP channel) |

### `distill-mcp setup`

Target a specific IDE:

| Flag | Effect |
|------|--------|
| `--claude` | Configure Claude Code only |
| `--cursor` | Configure Cursor only |
| `--windsurf` | Configure Windsurf only |
| `--antigravity` | Configure Antigravity only |

Install opt-in extensions (introduced in `v0.10.0`):

| Flag | Effect |
|------|--------|
| `--hooks` | Install the pre-commit hook in the current project |
| `--install-precompact-hook` | Add the PreCompact hook into `~/.claude/settings.json` (idempotent, atomic via tempfile + rename) |
| `--uninstall-precompact-hook` | Cleanly remove the PreCompact entry |
| `--install-agent` | Copy the `distill-compressor.md` template into `~/.claude/agents/` |
| `--uninstall-agent` | Remove the installed agent |

Cross-cutting options:

| Flag | Effect |
|------|--------|
| `--dry-run` | Print the changes without mutating the filesystem |
| `--user-dir=<path>` | Override `HOME` for locating `settings.json` (useful in tests) |
| `--force`, `-f` | Overwrite an existing configuration even if it differs from the template |

See the [marker + PreCompact hook page](/docs/compressed-marker) for integration details.

### `distill-mcp doctor`

No flags. Checks Node version, `distill-mcp` availability, detected-IDE configuration, and the state of installed hooks.

### `distill-mcp analyze`

| Flag | Effect |
|------|--------|
| `--patterns`, `-p` | Glob patterns to scan (default `**/*.{ts,tsx,js,jsx,py,go,rs}`) |
| `--threshold`, `-t` | Token warning threshold (default `2000`) |
| `--json`, `-j` | Emit JSON instead of the text report |
| `--output`, `-o` | Write the report to a file instead of stdout |
